# User Manual: AI Forge Schema

## 1. Creating Agents (The Forge)
1.  Click the **Cyan Factory Icon** (`precision_manufacturing`) in the toolbar.
2.  **Define Signature**:
    *   **Inputs**: Add fields like `user_query`, `database_schema`.
    *   **Logic**: Add reasoning steps (Chain of Thought, ReAct).
    *   **Outputs**: Define the JSON schema or text format.
3.  **Compile**: Watch the Live Preview update with XML tags/
4.  **Export**: Click "Export to Editor" to move the agent to the main workspace.

## 2. Securing Agents (The Sentinel)
1.  Click the **Red Shield Icon** in the toolbar.
2.  Click **INITIATE AUDIT**.
3.  Wait for the system to run attacks (Translation, Completion Injection).
4.  **Review**: See Pass/Fail status for each vector.
5.  **Patch**: If vulnerable, click "Deploy Countermeasures" to inject security protocols.

## 3. Advanced Templating (The Thinker)
1.  Click the **Puzzle Icon** in the toolbar.
2.  Browse the "Template Library".
3.  Select a robust template (e.g., "Senior Code Architect" or "XML Agent Structure").
4.  The editor will be populated with the template.

## 4. Testing & Verification
*   **Battle Arena**: Use the "Swords" icon to compare your current prompt against a proposed variation.
*   **Evaluate**: Use the "Analytics" button to get a 0-100 quality score based on a test run.
